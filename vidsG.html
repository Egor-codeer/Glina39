Глину делят на виды по трем главным признакам:

1. По цвету (составу):

• 1. Белая (каолин) — самая чистая, для фарфора.

• 2. Красная - с железом.

• 3. Голубая, зеленая, черная — с разными минералами.

2. По «жирности» (пластичности):

• 1. Жирная - липкая, почти без песка (трескается при сушке).

• 2. Тощая - много песка или ила (плохо лепится, рассыпается).

3. По назначению:

• 1. Гончарная — идеальный баланс для лепки посуды.

• 2. Огнеупорная — держит экстремальный жар (для печей).

• 3. Бентонитовая — сильно разбухает (для бурения и очистки).
